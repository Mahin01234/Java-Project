## 📁 File Overview

| File Name        | Description |
|------------------|-------------|
| **NumberGuessingGame.java** | A simple number guessing game where the player guesses a random number between 1 and 100 within 5 attempts, receiving "Too high" or "Too low" hints after each guess. | 
| **TicTacToeSinglePlayer.java** | A console-based, single‑player Tic‑Tac‑Toe game where you play as 'X' against a computer opponent ('Y'). The game features a simple AI that prioritizes winning moves, then blocking your winning moves, and finally selects a random available cell. It also keeps a live scoreboard tracking your wins, computer wins, and draws across multiple rounds. |
